# Outcomes

What this system is for — the payoff that justifies the daily habits.

## Personal knowledge base
Everything you learn, decide, and produce lives in one structured, searchable place
you own. No more "I know I read this somewhere."

## AI-powered memory
`MEMORY.md` and the session logs give the agent durable context, so it picks up where
you left off instead of starting cold every time.

## Faster decision making
Retrieval and decision-support workflows sit on top of your own notes, so you decide
from what you actually know — quickly and with the receipts.

## Consistent workflows
Templates and documented routines mean repeatable tasks come out the same high quality
every time, whether you or the agent runs them.

## Compound intelligence
Logs, notes, skills, and memory accumulate. Each week of use makes the next week more
capable. The system's value grows with the inputs — that's the whole point.

---
> If a habit or file in this vault isn't serving one of these outcomes, prune it.
> The system should stay as small as it can while still delivering these.
