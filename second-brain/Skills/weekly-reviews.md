# Skill: Weekly Reviews

Run the weekly review with the agent as facilitator.

- **Trigger:** "Run my weekly review."
- **Inputs:** the past week's `Sessions/` logs, `PARA/`, `Memory/goals.md`.
- **Output:** a completed `Templates/weekly-review.md` note + an updated `MEMORY.md`.

## Steps
1. Confirm the inbox is empty; if not, run capture-processing first.
2. Pull wins, slips, and lessons from the week's session logs.
3. Walk each PARA project: still active? next action defined?
4. Compare progress to `Memory/goals.md`; propose re-prioritization.
5. Draft next week's top 3.
6. Apply system maintenance: update `MEMORY.md`, suggest one skill to build, flag
   cleanup (dupes, stale notes, broken links).

## Quality bar
- Ends with a concrete plan for next week and an accurate `MEMORY.md`.
